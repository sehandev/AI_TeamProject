# AI_TeamProject
2021년 홍익대학교 4학년 1학기 인공지능 기말 프로젝트

팀원 : 김성돈, 박세연, 이도연, 정소윤

사람의 눈으로 구분하기 힘들어보이는 재규어, 표범, 치타를 구분하는 RNN(LSTM, GRU), CNN(VGGNet, GoogLeNet, ResNet) 제작

Data : ImageNet dataset에서 필요한 class의 data만 따로 추출

사용 기술
- Deep Learning
  1. pytorch
  2. pytorch-lightning
  3. ray[tune]
- DevOps
  1. Docker
  2. Docker Compose
